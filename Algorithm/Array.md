## 122. 买卖股票的最佳时机 II
求单调增的子序列，每次都买在最低点，卖在最高点，利润就是最大的。
下面还有个最优解，实际上简化了我的思路，因为连涨可以连续买，跌了就把跌之后的股票算作left。
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {   
        int ans = 0;
        int n = prices.size();
        for (int i = 1; i < n; ++i) {
            ans += max(0, prices[i] - prices[i - 1]);
        }
        return ans;
    }
};
```

## 55. 跳跃游戏

## 45. 跳跃游戏 II
方法1. 逆向维护一个posi，初始化为 size-1，从后向前遍历，找到了能跳到posi的最小的i，更新posi为i
方法2. 维护区间内能到的最远位置为`end`，当前能到的最远位置为`right_most`，每次遍历到`end`时，更新`end`为`right_most`，并增加步数
